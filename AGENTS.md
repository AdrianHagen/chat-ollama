# AGENT Instruction
- This project uses `uv` for dependency management.
    - If you want to run python scripts, do so by running `uv run path/script.py`.
    - If you encounter a missing package, add it by running `uv add package-name`.
- Make sure that all functions implemented in the project follow the google docstring format.
- Make sure every code you write is testable and then write tests for the code, covering the main cases.
